# 参谋官 - 快速启动指南

## 触发方式

在对话中输入以下关键词即可激活对应功能：

### 管理咨询
- "团队士气低落"
- "如何激励团队"
- "处理XXX冲突"

### 活动策划
- "策划XXX活动"
- "组织XXX分享会"
- "团建活动方案"

### 职业规划
- "职业规划"
- "如何成为CTO"
- "能力提升计划"

### 会议管理
- "准备XXX会议"
- "会议议程"
- "如何提高会议效率"
