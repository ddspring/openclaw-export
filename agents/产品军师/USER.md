# USER.md - 服务对象

## 用户画像

- **角色**：嵌入式软件部门经理（兼顾产品职责）
- **行业**：工业通信与嵌入式系统
- **产品**：工业以太网交换机、TSN交换机、工业通信设备

## 核心需求

1. **市场洞察**：了解行业动态、技术趋势、市场机会
2. **竞品分析**：分析竞争对手的产品特性、市场策略
3. **产品规划**：制定产品路线图、版本规划、需求优先级
4. **产品决策**：新功能是否值得投入、目标市场选择

## 工作场景

- 产品战略规划（年度/季度）
- 竞标准备（分析竞品优势）
- 客户需求评估（优先级排序）
- 版本规划（功能规划、资源分配）

## 关注领域

- 工业通信市场动态（TSN、工业以太网）
- 主要竞品分析（东土、三旺、迈威、 Hirschmann、Moxa等）
- 客户需求痛点
- 行业认证和标准

## 沟通偏好

- 喜欢数据支撑的分析
- 需要清晰的对比表格
- 重视市场导向的建议
- 关注可执行性

## 约束条件

- 需要平衡技术可行性和市场价值
- 考虑公司资源和团队能力
- 产品需要满足工业级可靠性
