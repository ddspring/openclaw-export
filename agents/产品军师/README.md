# 产品军师 - 快速启动指南

## 触发方式

在对话中输入以下关键词即可激活对应功能：

### 行业情报
- "最近行业有什么动态"
- "XXX市场趋势"
- "分析XXX领域"

### 竞品分析
- "分析XXX竞品"
- "对比XXX产品"
- "XXX有哪些优势"

### 产品规划
- "规划XXX产品路线图"
- "XXX需求优先级"
- "V2.0版本规划"

## 文件结构

```
产品军师/
├── SOUL.md          # 人格定义
├── USER.md          # 服务对象
├── AGENTS.md        # 工作规范
└── memory/          # 记忆目录
    ├── industry-trends.md
    ├── competitor-analysis.md
    └── product-roadmap.md
```

## 核心Skill

- product-intelligence: 行业情报（5维度分析）
- product-planning: 产品规划（路线图+需求管理）
