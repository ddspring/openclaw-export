# 普速铁路通信网络方案调研报告

**调研日期**：2026年3月11日  
**调研范围**：400M数传电台应用、车地无线通信方案、典型组网方案

---

## 执行摘要

普速铁路无线通信正处于从模拟向数字化转型的关键阶段。2026年1月，中国通号自主研发的400MHz数字无线列调系统（DRTD）在广铁集团益湛铁路益阳至娄底段正式投入运营，标志着国内普速铁路无线列调技术首次实现数字化升级。本报告围绕三大核心问题进行系统梳理。

---

## 一、400MHz数传电台在铁路中的应用

### 1.1 典型产品与厂家

| 产品/系统 | 厂家 | 应用场景 |
|-----------|------|----------|
| 400MHz数字无线列调系统（DRTD） | 中国通号（研究设计院集团+通信信息集团） | 广铁集团益湛铁路益娄段 |
| WTZJ-Ⅲ型CIR机车综合无线通信设备 | 中国铁道科学研究院 | 高速/普速机车 |
| GD400HH高速数传电台 | 深圳固迪通信 | 工业数传 |
| 400MHz集群系统 | 各铁路通信设备厂商 | 站场调车 |

### 1.2 关键技术参数

| 参数类别 | 具体指标 |
|----------|----------|
| **工作频段** | 457.2–458.65 MHz / 467.2–468.65 MHz（上行/下行） |
| **信道带宽** | 25 kHz |
| **发射功率** | 基站：≤50W；移动终端：≤5W |
| **数据速率** | 9600/19200bps（空中速率） |
| **通信模式** | 个呼、组呼、广播；支持直通模式（DMO）与中继模式（RMO） |
| **定位依赖** | 北斗定位实现自动切换与位置寻址 |

### 1.3 铁路应用场景

1. **列车调度指挥**：行车调度语音通信、调度命令无线传送、进路预告传送
2. **车次号校核**：列车无线车次号校核信息传送
3. **货列尾中继**：货物列车尾部风压数据中继传输
4. **站场调车**：平面调车业务、无线对讲业务
5. **地方铁路**：适用《地方铁路400MHz数字无线通信系统技术条件》（T/CZGJ002—2025）

### 1.4 系统特点

- **多制式兼容**：车载终端支持450MHz / 400MHz / GSM-R多种制式，可根据北斗卫星定位自动切换通信制式
- **语音与数据双时隙并发**：支持语音通信与实时数据业务同时传输
- **加密通信**：支持加密与抗干扰功能
- **新旧系统兼容**：实现与既有450MHz模拟系统平稳衔接

## 二、车地无线通信方案

### 2.1 普速铁路主流无线通信制式对比

| 制式 | 频段 | 速率 | 主要应用 | 优劣势 |
|------|------|------|----------|--------|
| **450MHz模拟列调** | 450MHz | 低 | 既有普速铁路 | 成本低但功能单一，面临淘汰 |
| **400MHz数字列调（DRTD）** | 457-468MHz | 中 | 新建/改造普速铁路 | 数字化升级首选，兼顾既有设备 |
| **GSM-R** | 885-889MHz / 930-934MHz | 中 | 高速铁路/部分普速铁路 | 技术成熟，支持高速移动，但未全面覆盖 |
| **5G-R** | 1.8GHz+ | 高 | 下一代铁路 | 宽带业务承载，试点应用 |
| **LTE-R** | 1.8GHz | 高 | 枢纽/重载铁路 | 支持多媒体调度，融合业务 |

### 2.2 GSM-R vs 400MHz集群 vs 其他

**GSM-R系统特点**：
- 工作频段：上行885-889MHz，下行930-934MHz
- 支持最高500公里/小时移动速度下的稳定通信
- 具备组呼、广播、多优先级、紧急呼叫等功能
- 基于GSM和EIRENE-MORANE技术架构构建

**400MHz数字集群优势**：
- 绕射能力强，覆盖范围广，适合平原/郊区
- 设备成本相对较低，部署灵活
- 与既有450MHz系统兼容性好
- 特别适合没有GSM-R网络覆盖的既有普速铁路

**5G-R/LTE-R发展方向**：
- 支持高带宽业务（视频监控、乘客信息服务）
- 铁路5G-R无线接入网组网方案已有多项研究
- 适用于枢纽地区和高速移动场景

### 2.3 列车调度指挥的通信需求

| 业务类型 | 需求等级 | 承载方式 |
|----------|----------|----------|
| 行车调度语音 | 最高（安全关键） | 400MHz数传 / GSM-R |
| 调度命令传送 | 高（安全相关） | 400MHz数传 / GSM-R |
| 进路预告 | 高 | 400MHz数传 |
| 车次号校核 | 中 | 400MHz数传 |
| 列车尾部风压 | 中 | 400MHz数传 |
| 视频监控 | 中-高（未来） | 5G-R/LTE-R |

## 三、典型组网方案

### 3.1 区间通信

**覆盖方式**：
- **光纤直放站**：适用于长距离区间，通过光纤拉远覆盖
- **无线中继**：采用400MHz数字中继模式（RMO）延伸覆盖
- **隧道覆盖**：隧道内布置泄漏电缆或MESH基站

**技术特点**：
- 采用"小区合并"技术减少切换和干扰
- 基站间距通常3-5公里（视地形而定）
- 隧道口MESH基站与沿线基站通讯距离最远可达5-60公里

### 3.2 车站通信

**站场无线网络**：
- 采用400MHz集群系统集中设置
- 设备数量少，便于集中维护管理
- 支持站场平面调车、无线对讲、SAM系统车地信息传送

**枢纽地区**：
- 多制式并存（450MHz / 400MHz / GSM-R）
- 需考虑不同线路间的互连互通
- 优先考虑LTE宽带集群方案支撑大数据应用

### 3.3 枢纽通信

**网络架构**：
- 核心网：中国通号/铁科院核心交换中心
- 接入网：分布式基站（5G-R/LTE） + 集群基站（400MHz）
- 终端：机车综合无线通信设备（CIR）

**覆盖策略**：
- 红线外（区间）：分布式基站S11配置，2个扇区覆盖铁路线路
- 红线内（站场）：集群基站 + 室内分布系统
- 隧道/桥梁：泄漏电缆 + MESH自组网

## 四、关键信息点总结

### 4.1 技术演进趋势

1. **模拟→数字**：400MHz数字无线列调正在逐步替代450MHz模拟系统
2. **窄带→宽带**：5G-R/LTE-R开始试点，承载视频等宽带业务
3. **单制式→多制式融合**：车载终端多模化，支持自动切换

### 4.2 决策建议

| 场景 | 推荐方案 | 理由 |
|------|----------|------|
| 新建普速铁路 | 400MHz数字列调 | 兼顾成本与功能，国产化程度高 |
| 既有线改造 | 400MHz数传 + 兼容450MHz | 平滑过渡，减少投资 |
| 高速铁路 | GSM-R / 5G-R | 技术成熟，支持高速移动 |
| 枢纽/站场 | LTE宽带集群 | 支撑多媒体调度和大数据应用 |

### 4.3 风险与挑战

- **频率资源**：400MHz频段带宽有限，难以承载高带宽业务
- **覆盖盲区**：隧道、弯道等特殊地形需额外投资
- **标准统一**：多制式并存带来互操作性挑战
- **国产化**：核心芯片和关键器件仍需关注供应链安全
