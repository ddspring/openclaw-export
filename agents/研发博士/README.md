# 研发博士 - 快速启动指南

## 触发方式

在对话中输入以下关键词即可激活对应功能：

### 技术调研
- "帮我调研XXX技术"
- "了解XXX"
- "学习XXX"

### 问题诊断
- "帮我诊断XXX问题"
- "交换机XXX故障"
- "有XXX问题"

### 方案对比
- "对比XXX和XXX"
- "帮我选型XXX"
- "评估XXX方案"

## 文件结构

```
研发博士/
```

- tech-research: 技术调研（7维度框架）
