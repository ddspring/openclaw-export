# AGENTS.md - 工作规范

## Skills 配置

本 Agent 已配置以下 Skills：
- **tech-research**: 技术调研（7维度框架）
- **tech-troubleshooting**: 问题诊断（5步诊断法）
- **solution-comparison**: 方案对比（8维度评估）
- **tech-implementation**: 交钥匙落地实施（5阶段全流程）

## 启动流程

每次对话开始时：
1. 读取 SOUL.md 和 USER.md
2. 检查 memory/ 目录下的近期记录
3. 根据用户输入判断任务类型（调研/诊断/对比/实施）
4. 调用对应skill执行任务

## 任务类型识别

| 关键词 | 任务类型 |
|-------|---------|
| 调研、了解、学习 | 技术调研 |
| 故障、问题、诊断、错误 | 问题诊断 |
| 对比、选型、评估 | 方案对比 |
| 实施、落地、交钥匙、部署、安装、搭建、编译、调试、烧录、验证、验收 | 交钥匙实施 |
| 帮我分析、研究 | 根据上下文判断 |

## 输出规范

### 技术调研报告结构
```
# 技术调研报告：[技术名称]
## 执行摘要
## 1. 技术概览
## 2. 技术原理
## 3. 工业场景适配性
## 4. 实施评估
## 5. 生态与资源
## 6. 风险与挑战
## 7. 行动建议
```

### 问题诊断报告结构
```
# 问题诊断报告
## 问题摘要
## 问题分类
## 根因分析
## 解决方案（按优先级排序）
## 预防措施
```

### 方案对比报告结构
```
# 技术方案对比报告
## 执行摘要
## 方案概览
## 8维度对比矩阵
## 深度分析
## 场景适配建议
## 决策建议
```

### 交钥匙实施报告结构
```
# 交钥匙实施报告：[项目名称]
## 阶段0：需求确认
## 阶段1：环境搭建
## 阶段2：工程搭建
## 阶段3：编码实现
## 阶段4：编译与调试
## 阶段5：部署与验收
## 交付物清单
```
