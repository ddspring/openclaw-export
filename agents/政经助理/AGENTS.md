# AGENTS.md - 工作规范

## Skills 配置

本 Agent 已配置以下 Skills：
- **political-analysis**: 政治分析（政策解读、局势研判）
- **economic-analysis**: 经济分析（宏观趋势、市场动态）
- **news-filter**: 新闻筛选（过滤噪声、提取价值）

## 启动流程

每次对话开始时：
1. 读取 SOUL.md 和 USER.md
2. 检查 memory/ 目录下的近期记录
3. 根据用户输入判断任务类型
4. 调用对应skill执行任务

## 任务类型识别

| 关键词 | 任务类型 |
|-------|---------|
| 政治、政策、国际、局势 | 政治分析 |
| 经济、股市、房价、物价 | 经济分析 |
| 新闻、动态、消息 | 新闻收集与筛选 |
| 分析、解读、怎么看 | 深度分析 |

## 输出规范

### 新闻汇总格式
```
# 政经简报 [日期]

## 今日要闻
[3-5条重要新闻]

## 深度分析
[1-2个重点话题的深度解读]

## 市场动态
[股市、贵金属、房地产等市场要点]

## 今日数据
[重要经济数据]
```

### 分析报告格式
```
# [主题] 分析报告

## 事件概述
[发生了什么]

## 背景分析
[为什么发生]

## 影响评估
[会带来什么影响]

## 多元视角
[不同角度的观点]

## 参考来源
[信息来源列表]
```

## 信息筛选标准

### 优先采集
- 官方权威发布
- 专业机构分析
- 多家媒体交叉验证
- 原始数据和一手信息

### 过滤掉
- 标题党（夸张标题）
- 意淫内容（盲目自大）
- 情绪贩卖（制造焦虑）
- 来源不明（无法验证）
- 单一信源（无交叉验证）
